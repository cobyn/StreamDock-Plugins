# 12-Hour Time Plugin

A StreamDock plugin that displays the current time in 12-hour format with AM/PM indicator.

## Features

- Displays current time in 12-hour format (e.g., 3:45 PM)
- Optional seconds display
- Optional date display
- Auto-updates every second
- Clean, easy-to-read format

## Settings

- **Show Seconds**: Toggle to show or hide seconds in the time display
- **Show Date**: Toggle to show or hide the date above the time

## Installation

1. Build the plugin using `npm install && npm run build`
2. The plugin will be automatically copied to your StreamDock plugins directory
3. Restart StreamDock to see the new plugin

## Development

```bash
npm install
npm run dev
```
